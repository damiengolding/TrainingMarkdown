Web Security Testing Guide (WSTG) - Web Application Security Testing

How to prime a ZAP session (fits after 1. Information Gathering and really)

Initialize ZAP

Set target and initialise sitemap

  1. Make sure mode is set to 'Protected'
  2. Make sure that the 'Forced user' toolbar button is enabled
  3. Make sure you identify and exclude any logout pages ASAP: once found, right click and select "Exclude from -> Scanner"
  4. Start traditional spider from bottom tab (open a 'Spider' tab if necessary)
  5. Add target site to context
  6. Use Tools -> Authentication test to add a user to the context
  7. Select the user in the context as 'Forced user'
  8. Start AJAX spider (use chrome headless)
  9. Use 'Forced browsing' to add to the site map
